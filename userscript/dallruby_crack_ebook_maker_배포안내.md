# dallruby crack e-book maker 배포 안내

## 배포 파일

배포할 파일은 이것 하나입니다.

`dallruby_crack_ebook_maker_v10.user.js`

달루비 헤더 이미지는 스크립트 안에 작게 내장되어 있어서 이미지 파일을 따로 올리지 않아도 됩니다.

## GitHub 배포 방법

1. GitHub에서 새 repository를 만듭니다.
2. repository 이름 예시: `dallruby-crack-ebook-maker`
3. `dallruby_crack_ebook_maker_v8.user.js` 파일을 업로드합니다.
4. 업로드한 파일을 열고 `Raw` 버튼을 누릅니다.
5. Raw 주소를 사람들에게 공유합니다.

Raw 주소 예시는 이런 형태입니다.

```txt
https://raw.githubusercontent.com/깃허브아이디/dallruby-crack-ebook-maker/main/dallruby_crack_ebook_maker_v10.user.js
```

Tampermonkey가 설치된 사람이 이 Raw 링크를 열면 설치 화면으로 이동합니다.

## 유저용 설치 안내

1. Chrome에 Tampermonkey를 설치합니다.
2. 크랙에 로그인합니다.
3. 배포자가 준 Raw 링크를 엽니다.
4. Tampermonkey 설치 화면에서 `설치`를 누릅니다.
5. `crack.wrtn.ai`를 새로고침합니다.
6. 오른쪽 위 `dallruby crack e-book maker` 패널에서 `현재 채팅 저장하기`를 누릅니다.

## 사용법

보통은 저장할 채팅방을 연 뒤 `현재 채팅 저장하기`만 누르면 됩니다.

- `마크다운 .md 백업`: 글 백업용 파일입니다. 메모장, Obsidian, Notion, VS Code 등에서 열기 좋습니다.
- `전자책 .epub 이북`: 폰의 전자책 앱에서 책처럼 읽는 파일입니다.
- `다시 잡기`: 큰 방이나 오래 열린 페이지에서 채팅방 인식이 꼬였을 때만 누릅니다.
- `문제 해결용 정보`: 오류가 났을 때 배포자에게 보낼 API 로그입니다. 평소에는 안 봐도 됩니다.

## 주의

- 이 도구는 각자 자기 브라우저 로그인으로 자기 채팅을 저장합니다.
- 다른 사람의 채팅을 가져오는 도구가 아닙니다.
- 크랙 사이트/API 구조가 바뀌면 업데이트가 필요할 수 있습니다.
